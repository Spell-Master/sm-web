
Esse é um exemplo mais simples possível para enviar e-mail's
usando a minha classe "Mailer" que por sua vez usa a classe
"PHPMailer"

==================================================================
= Como configurar
==================================================================
Vá ao arquivo "configuracao.php" para configurar os acessos.
No próprio arquivo estará descrito como você deve preencher
os dados.

==================================================================
= Usando gratuitamente um servidor SMTP
==================================================================
Se você optar por utilizar o SMTP do G-MAIL assim como eu fique
sabendo que você terá que permitir o acesso.
Pois a google protege sua conta, assim sendo sempre que um
terceiro tentar acessar-la será bloqueado.
Para esse desbloqueio (o caso do gmail) acesse o link:
https://myaccount.google.com/security
Logado com a conta que você configurou o mailUser no
configuracao.php

==================================================================
= Testando sem hospedar em um servidor
==================================================================
Você pode usar até mesmo o servidor local "localhost" para
executar porém para isso você terá que configurar seu servidor
local aí senguindo os passos abaixo:
"Não irei informar o caminho para os arquivos por desconhecer qual
a matrix do seu sistema"

1 - Vá até o diretório do seu sistema "Apache"
2 - Abra com um editor o arquivo "httpd.conf"
3 - Procure pela linha "LoadModule ssl_module modules/mod_ssl.so"
4 - Remova o "#" isso para habilitar o modulo.
4.2 - Possivelmente esse modulo já vai está habilitado, depende
de qual sistema apache está usando.
* É provável que não encontrará algumas configurações a seguir *
* Depende muito do qual sistema está usando *
5 - Vá até o diretório do seu sistema "PHP"
6 - Abra com um editor o arquivo "php.INI"
7 - Remova o ";" nas extensões a seguir:
      php_curl
      php_openssl
      php_sockets
      php_smtp
7.2 - Se não encontrou uma ou mais é porque elas já estão ativas
pela necessidade do sistema por elas.
8 - Reinicie os sistemas e pronto.

Caso esteja em uma máquina windows e esteja usando um software
anti-vírus cujo o mesmo é de baixa qualidade não dando suporte
ao um monitoramento ativo do firewall.
Ele pode interromper a conexão, fazendo o script não funcionar.
Para que funcione você pode parar a proteção do mesmo
ou pagar por um software decente :)
