trigger

Aciona ouvintes de eventos adicionados executando-os.
Quaisquer eventos adicionados por addEventListener também são afetados pelo método trigger 
<button id="exemplo">Exemplo</button>

<script>
    var exemplo = jsd('#exemplo');

    exemplo.on('click', function (e) {
        console.log('Evento click');
    });
    exemplo.on('mousedown', function () {
        console.log('Evento mousedown');
    });
    exemplo.on('dblclick', function () {
        console.log('Evento dblclick');
    });
</script>
Executa evento expecífico 
exemplo.trigger('click');
Executa todos eventos 
exemplo.trigger();
