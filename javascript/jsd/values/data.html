data

Obtem valores do pseudo atributo data 
<div id="exemplo" data-exemploA="Valor A" data-exemploB="Valor B" data-exemploC="Valor C">Exemplo</div>
Obtem todos dataset 
jsd('#exemplo').data()
Obtem o valor do dataset expecífico 
jsd('#exemplo').data('data-exemploB');
// Ou apenas a definição
jsd('#exemplo').data('exemploB')
