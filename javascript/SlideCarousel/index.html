Utilização

Adicione os arquivos necessários:

<link href="SlideCarousel.css" rel="stylesheet" type="text/css"/>
<script src="SlideCarousel.js" type="text/javascript"></script>

Crie a estrutura html:

<div id="identificador">
    <div class="slide-item">Slide 1</div>
    <div class="slide-item">slide 2</div>
    <div class="slide-item">Slide 3</div>
<div>

Você deve criar uma tag e adicionar o atributo id, o valor desse atributo você fornece ao SlideCarousel para que ele saiba que esse é o elemento alvo ao qual ele deve interagir.

Dentro desse elemento você então adiciona os slides que deseja.

Cada slide deve possuir o classificador class="slide-item" ao qual o SlideCarousel saberá que esses são cada item pertecente.

ATENCÃO!

Qualquer outro elemento, texto, etc que esteja dentro do elemento alvo que não contenha o classificador slide-item será removido do documento.

Inicie em um javascript ao final do documento criando um objeto que instância a classe SlideCarousel.

Esse objeto poderá ser usado para acessar os demais métodos da classe.

<script>
    var objeto = new SlideCarousel('identificador');
</script>

Parâmetros opcionais

Qualquer desses parâmetros é opcional

Use somente aqueles que achar necessário.

Esses vão definir como o slide deve se comportar.

maxWidth: Você deve informar um número, esse número será a largura máxima em pixel's ao qual o slide terá.

Quando não definido a largura máxima será a largura real do elemento alvo.

autoPlay: Você deve informar um número, esse número definirá de quantos em quantos segudos cada item deve ser trocado.

Por padrão essa funcionabilidade estará desabilitada.

O valor mínimo é 3 "três" (segundos) abaixo desse valor resultará nos mesmos 3.

enableTouch: Você deve informar "false" para desabilitar.

Se habilitado permite que o usuário segure e arraste o item atual para trocar de slide (para segurar com o mouse e arrastar de dispositivos touch-screen).

Por padrão essa funcionabilidade estará ativa.

enableNav: Você deve informar "false" para desabilitar.

Se habilitado cria botões laterais para mudar o slide.

Por padrão essa funcionabilidade estará ativa.

Exemplo: 
var exemplo = new SlideCarousel('elemento_id', {
    maxWidth: 600, // largura máxima será de 600 pixel
    autoPlay: 4, // a cada 4 segundos troca o slide atual
    enableTouch: false, // não permite arrastar
    enableNav: false // não cria os botões de navegação
});

Métodos extra

next: Move o item atual para a esquerda e coloca o próximo no meio.

prev: Move o item atual para a direita e coloca o anterior no meio.

get: Obtem o elemento da instância do slide.

Exemplo: 
var exemplo = new SlideCarousel('elemento_id');
function proximo() {
    exemplo.next();
}
function anterior() {
    exemplo.prev();
}
function elementoAlvo() {
    exemplo.get();
}
