index

Seleciona o índice expecífico dos anexos a jsd

A função index fixa em um elemento dentre vários outros já instânciados também em outra camada da instância da jsd.

Para fins de manipulação distinta em apenas um elemento dos vários possíveis

<div class="exemplo">exemplo 1</div>
<div class="exemplo">exemplo 2</div>
<div class="exemplo">exemplo 3</div>
<script>
var exemplo = jsd('.exemplo');
console.log(
    exemplo.index(0), // exemplo 1
    exemplo.index(1), // exemplo 2
    exemplo.index(2), // exemplo 3

    exemplo[0].innerText, // Texto do exemplo 2
    exemplo.index(1)[0].innerText, // Texto do exemplo 2
    exemplo.text(), // Texto contido em todos div.exemplo
    exemplo.index(1).text() // Texto contido do exemplo 2
);
</script>
