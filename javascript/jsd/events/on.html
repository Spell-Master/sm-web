on

Adiciona ouvintes de eventos.
NOTA*

Para melhor uso e manipulação não é possível adicionar mais de um ouvinte do mesmo tipo no mesmo elemento.

Se por exemplo adicionar um ouvinte para submit não pode-se adicionar outro submit sem que remova a função anterior.

No entanto não interfere em comando inline e nem em adesões externas pelo addEventListener.

<button id="exemplo">Exemplo</button>
Adicionar função anônima 
// Ouvinte para um click
jsd('#exemplo').on('click', function (e) {
    exemplo.on('click', function (evento) {
    console.log('Função anônima');
    console.log(evento);
    console.log(this);
});
Adicionar função nomeada 
function funcaoDefinida() {
    console.log('funcaoDefinida');
    console.log(evento);
    console.log(this);
}

// Ouvinte para o mouse pressionado
jsd('#exemplo').on('mousedown', funcaoDefinida);
