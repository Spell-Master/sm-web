Observação:

Perceba ao visualizar o código fonte o uso do atributo data-modal.

Esse atributo é definido pelo ModalShow.js e não deve ser definido antecipadamente... Resumindo não adicione esse atributo!

Utilização

Adicione o arquivos necessários

<script src="ModalShow.js" type="text/javascript"></script>
<link href="ModalShow.css" rel="stylesheet" type="text/css" />

Crie a estrutura HTML

<div class="modal" id="identificar_modal">
    <div class="modal-box">
        <div class="modal-header"></div>
        <div class="modal-content">
            O conteúdo do modal aqui dentro...
        </div>
    </div>
</div>
Deve-se seguir essa estrutura, caso contrário erros indesperados pode ocorrer.

Mas o mais importante que a caixa pai tenha class="modal" e um atributo #ID

Entendemos que você pode ter mais de um modal no documento, e é pelo #id que vamos distinguir um do outro.

Crie a instância da classe ModalShow adicionando um objeto, então você informa como parâmetro o #ID do modal alvo.

<script>
    var exemplo = new ModalShow('identificar_modal');
</script>

Então para cada modal que deseja um novo objeto instanciando a class ModalShow deve ser iniciado.

Com esse objeto é vamos ter acesso ao métodos.

Métodos

open: Esse método abre a janela. Ele possui um parâmetro que define as característica do modal

Falarei mais a frente sobre as opções de forma detalhada.

close: Esse método fecha a janela.

title: Esse método define/altera o título da janela.

Como parâmetro você informa o texto que terá o título.

showX: Esse método mostra o botão de fechar quando ele não estiver ativo.

hiddenX: Esse método esconde o botão de fechar quando ele estiver ativo.

content: Esse método define/altera o conteúdo do modal.

Como parâmetro você informa o texto que terá o conteúdo.

Obs.: Você pode informar uma estrutura html também

<script>
    var exemplo = new ModalShow('identificar_modal'); // Instância criada no objeto "exemplo"
    exemplo.open(); // Abre o modal
    exemplo.close(); // Fecha o modal
    exemplo.title('Olá mundo'); // O título será: Olá mundo
    exemplo.showX(); // Mostra o botão de fechar
    exemplo.hiddenX(); // Esconde o botão de fechar
    exemplo.content('<div class="hello">Olá</div>'); // Altera o conteúdo
</script>

Opções

Essas são as opções ao qual você pode está usando quando for abrir o modal. Qualquer uma delas é opcional, use conforme achar necessário.

title: Nessa opção você informa um texto que terá o modal quando ele for aberto.

close: Nessa opção você informa "true" ou "false".

Se informar como false, o botão de fechar será oculto, no entando se você não informar como true ou mesmo se não informar o botão, será exibido de qualquer forma.

width: Nessa opção você informa um número ao qual determinará a largura máxima do modal.

effect: Nessa opção você informa um texto com o tipo de efeito que deve ser executado toda vez que o modal for aberto.

Os seguintes efeitos estão disponíveis:

  • fade Esse efeito faz com que o modal apareça lentamente.
  • zoom Esse efeito faz com que o modal começe pequeno até o seu tamanho real.
  • top Esse efeito faz com que o modal venha pelo topo de fora da tela até sua posição real.
  • bottom Esse efeito faz com que o modal venha por baixo de fora da tela até sua posição real.
  • left Esse efeito faz com que o modal venha pela esquerda de fora da tela até sua posição real.
  • right Esse efeito faz com que o modal venha pela direita de fora da tela até sua posição real.

onOpen: Nessa opção você informa uma função de deve ser executada toda vez que o modal é aberto.

onClose: Nessa opção você informa uma função de deve ser executada toda vez que o modal é fechado. 

objeto.open({
    title: 'Título da Janela',
    close: true,
    width: 600,
    effect: 'fade',
    onOpen: function () {
        // fazer alguma coisa quando o modal for aberto
    },
    onClose: function () {
        // fazer alguma coisa quando o modal for fechado
    }
});
