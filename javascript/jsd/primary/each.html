each

Percorrendo elementos anexos a jsd
A função each quando recursiva a instância de um elemento trabalha de forma similar a função motora each da jsd.

Sua ação está em manipular os elementos e todas as chaves do objeto.

<div class="exemplo">exemplo 1</div>
<div class="exemplo">exemplo 2</div>
<div class="exemplo">exemplo 3</div>

<script>
    jsd('div.exemplo').each(function (valor, indice) {
        console.log(
            valor,  // O elemento anexo na instância
            indice, // O número da chave da instância
            this    // O elemento não anexado na instância
        );
    });
</script>
