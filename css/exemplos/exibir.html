IMPORTANTE

Todas propriedades a seguir utilizam o atributo display do elemento.

Não devem ser mescladas

Listagem

list

<p class="list">a</p>
<p class="list">b</p>
<p class="list">c</p>

Essa propriedade vai transformar o elemento alvo um índice de lista desordenada como se fosse a tag <li> dentro de uma <ul>.

list-none

<ul class="list-none">
    <li>a</li>
    <li>b</li>
    <li>c</li>
</ul>

Excluzivo para as tag's <ul> e <ol>, essa propriedade remove o ícone de marcação para lista ordenada ou desordenada.

Renderização em caixa de bloco

block

show

Faz com que um elemento gere uma caixa de bloco.

Também pode ser usado para exibir um elemento oculto, uma vez que o elemento deixa de ser ignorado e passa ser uma caixa de bloco.

<div class="block"></div>
<div class="show"></div>

none

hide

Faz com que um elemento não gere caixas na estrutura de formatação.

Mesmo que elas estejam presentes não são visíveis e não tendo efeito nenhum sobre a estrutura do layout.

<div class="none"></div>
<div class="hide"></div>

Por exemplo se eu declarar que um elemento tenha o classificador block:

Ele passa ser um bloco.

E se por acaso eu declarar um elemento tenha o classificador none.

Ele estará na arquitetura da renderização, porém na visualização do documento é como se ele não existisse.

Declarações incoerentes não são possíveis, digamos as seguintes definições:

<div class="none block"></div>
<div class="block none"></div>

Perceba que no primeiro elemento declarei none depois block e no segundo declarei ao inverso.

Ou seja quero que seja bloco, mas oculto; o que não faz nenhum sentido.

Em ambos os casos os elementos serão ocultos, tornando a coisa mais sensata.

No entanto, muitas ás vezes para uma integração com o usuário, por alguma razão geralmente por script um elemento que estar oculto deve ser mostrado, ou virse-e-versa.

Podemos aplicar o show / hide, aos quais anularão o block / none

Exemplo:

<div class="none show">Mesmo tendo delcarado "none"(invisível) o elemento é uma caixa bloco (visível).</div>
<div class="block hide">Mesmo tendo delcarado uma caixa bloco "block" (visível) o elemento é (invisível).</div>

Assim sendo você pode usar:

  • show para anular none
  • hide para anular block
