Alinhamento de colunas

  • row
  • row-pad

A propriedade row somente irá alinhar lado a lado cada coluna.

A propriedade row-pad terá o mesmo efeito de seu irmão row, mas faz com que cada coluna ganhe um espaçamento interno. Ou seja tudo que estiver dentro de uma coluna se distanciará das margens da linha.

Para melhor entendimento prestamos atenção ao prefixo col- logo seguindo do valor a ser aplicado.

Cada elemento pai definimos uma linha com row ou row-pad então se aplicamos quantas partes essa linha deve ser dividida.

E quando não houver disponibilidade de tela para conter a quantidade de colunas ao qual dividimos a linha, cada coluna salta para um espaço abaixo.

Espaçamento

Sabendo que se definir a linha como row-pad vamos criar um distaciamento da coluna às margens da linha.

Esse espaçamento é determindado pela configuração --spacing-medium, você poderá alterar essa configuração para ajustar o espaçamento conforme desejar.

Porém aterar essa configuração irá intervir nos demais padrões que a usam, então você pode está aplicando, o próprio conceito de espaço interno padrão para cada coluna.

É tudo questão de matemática

Sabendo que cada linha se divide de acordo com a(s) coluna(s) aplicada podemos criar um layout usando a matemática.

Então podemos criar o layout responsivo com infinitas possibilidades.

Vejamos abaixo como funciona a coisa toda na prática:

<div class="row">
    <div class="col-single bg-dark-grey">
        <div class="box-y-50"></div>
    </div>
    <div class="col-single bg-black">
        <div class="padding-all box-y-200"></div>
    </div>
</div>
<div class="row-pad">
    <div class="col-twothird bg-grey">
        <div class="row-pad">
            <div class="col-3"><div class="bg-red box-y-100"></div></div>
            <div class="col-3"><div class="bg-green box-y-100"></div></div>
            <div class="col-3"><div class="bg-blue box-y-100"></div></div>
            <div class="col-3"><div class="bg-yellow box-y-100"></div></div>
            <div class="col-3"><div class="bg-pink box-y-100"></div></div>
            <div class="col-3"><div class="bg-purple box-y-100"></div></div>
            <div class="col-1">
                <div class="row">

                </div>
            </div>
        </div>
