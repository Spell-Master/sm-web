SelectOption.js

Apresentação:

SelectOption

Utilização

Adicione os arquivos necessários:

<link href="SelectOption.css" rel="stylesheet" type="text/css"/>
<script src="SelectOption.js" type="text/javascript"></script>

Crie a estrutura html:

<select data-select="">
    <option value="">Opção</option>
    <option value="">Opção</option>
    <option value="">Opção</option>
</select>

A estrutura HTML é muito simples, apenas uma ou mais tag <select> definida com as opções e valores como você bem desejar, e sim você poderá ter várias, a única coisa necessária é o atributo data-select="", ao qual o SelectOption.js irá saber que essa será uma tag ao qual vai interagir, então para cada cada tag <select> ao qual queira a pesonalização adicione esse atributo.

Inicie em um javascript ao final do documento criando um objeto que instância a classe SelectOption.

Esse objeto poderá ser usado para acessar os demais métodos da classe.

<script>
    var objeto = new SelectOption();
</script>

Assim sendo não é necessário intervenções, o SelectOption irá buscar por todas as tag's <select> cujo possuam o atributo data-select.

Mas se no momento não houver nenhum elemento, ou fora removido, ou mesmo um novo adicionado, você poderá está usando o método extra.

Exemplo Prático

<select name="seletor" data-select="">
    <option value="">Selecionar Opção</option>
    <option value="valor_1">Opção 1</option>
    <option value="valor_2">Opção 2</option>
    <option value="valor_3">Opção 3</option>
    <option value="valor_4">Opção 4</option>
    <option value="valor_5">Opção 5</option>
    <option value="valor_6">Opção 6</option>
    <option value="valor_7">Opção 7</option>
    <option value="valor_8">Opção 8</option>
    <option value="valor_9">Opção 9</option>
</select>
<script>
    var select = new SelectOption();
</script>

Métodos extra

reload: Esse método recarrega todos seletores, então você poderá está usando para redefinir os dados nas seguintes situações:

Caso venha a adicionar/remover uma nova tag <select>

Altere alguma coisa em alguma tag <select> já iniciada.

Exemplo:

var objeto = new SelectOption(); // SelectOption iniciado sem nenhuma tag <select> no documento

function adicionarNovo() { // função para criar uma nova tag
    var novoSeleletor = document.createElement('select'), opcao, i = 0;
    novoSeleletor.name = 'seletor-adicionado[]';
    novoSeleletor.setAttribute('data-select', ''); // atributo necessário
    while (i !== 10) {
        i++;
        opcao = document.createElement('option');
        opcao.value = 'Opção criada ' + i;
        opcao.innerText = 'Opção criada ' + i;
        novoSeleletor.appendChild(opcao);
    }
    document.body.appendChild(novoSeleletor);
    objeto.reload(); // recarrega adicionando esse novo ao SelectOption
}

data: Esse método irá retornar os dados em um objeto dos seletores que estão sendo manipulados pelo SelectOption.

Os dados de retorno são:

  • index: Equivale ao índice no documento da tag <select>
  • name: Atributo name da tag <select>
  • value: Atual valor da tag <select>
  • select: A prória tag <select>

Exemplo:

var objeto = new SelectOption();
objeto.data();

Lembre-se que o SelectOption pode manipular várias seletores ao mesmo tempo, portanto quanto mais seletores estiverem sobre seus cuidados, maior será o objeto de retorno.
