Exemplo Prático

<div id="identificador">
    <ul class="tab-menu">
        <li class="tab-link">A</li>
        <li class="tab-link">B</li>
        <li class="tab-link">C</li>
        <li class="tab-link">D</li>
        <li class="tab-link">E</li>
    </ul>
    <div class="tab-body">Tab A</div>
    <div class="tab-body">Tab B</div>
    <div class="tab-body">Tab C</div>
    <div class="tab-body">Tab D</div>
    <div class="tab-body">Tab E</div>

<script>var tab = new TabPaginator('identificador');</script>

Utilização

Adicione os arquivos necessários:

<link href="TabPaginator.css" rel="stylesheet" type="text/css"/>
<script src="TabPaginator.js" type="text/javascript"></script>

Crie a estrutura html:

<div id="identificador">
    <ul class="tab-menu">
        <li class="tab-link">A</li>
        <li class="tab-link">B</li>
    </ul>
    <div class="tab-body">Tab A</div>
    <div class="tab-body">Tab B</div>
</div>

Antes da estrutura você deve criar um elemento pai, esse vai abrigar todo o resto, é importante que esse pai tenha o atributo #ID, ao qual você irá dizer ao TabPaginator que ele deve interagir com os elementos internos a ele.

Enquanto que a estrutura se trata de um elemento qualquer com o classfificador tab-menu

E dentro do tab-menu você cria a navegação entre as TAB, cada link de navegação deve possuir o classificador tab-link.

O próximo elemento é nada mais do que os conteúdos de cada link, esse elemento conteúdo deve possuir o classificador tab-body

É importante que siga essa estrutura para a coisa toda funcionar.

Você deve possuir a mesma quantidade de tab-link e tab-body Os links dentro do tab-menu e os tab-menu depois do menu.

Inicie em um javascript ao final do documento criando um objeto que instância a classe TabPaginator.

<script>
    var tab = new TabPaginator('identificador');
</script>

Então você fornece o #ID do local onde está a estrutura.

Parâmetro Opcional

Além do primeiro parâmetro da classe TabPaginator que é o id do local da estrutura (esse parâmento é obrigatório).

Existe um segundo parâmentro opcional, nele você informa qual o número da tab que deve ser aberta já no carregamento do script.

Exemplo:


// Segunda tab aberta ao invés da primeira no carregamento do script
var tab = new TabPaginator('identificador', 2);

Método extra

Você pode abrir qualquer tab por função externa como se o usuário clicasse em um link através do método openTab(numero da tab) exemplo: 
var tab = new TabPaginator('identificador');
tab.openTab(2); // Segunda tab aberta como se o usuário clicasse o segundo link
