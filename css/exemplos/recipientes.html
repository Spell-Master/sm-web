Centralizam e adaptam o conteúdo interno, mesmo que resolução seja menor ou maior que o esperado

Caso a largura da tela seja menor que o definido pelo recipiente o mesmo se ajusta para que seu tamanho limite seja a quantidade de pixel disponível

container-min

Recipiente de margens ajustadas até 480 pixels

container-sd

Recipiente de margens ajustadas até 640px pixels

container-medium

Recipiente de margens ajustadas até 1024px pixels

container-hd

Recipiente de margens ajustadas até 1280px pixels

container

Recipiente de margens ajustadas até 1920px pixels

container-2k

Recipiente de margens ajustadas até 2560px pixels

container-4k

Recipiente de margens ajustadas até 3280px pixels

container-max

Recipiente de margens ajustadas até 7680px pixels

A melhor forma para você ver como funciona os container's sendo aplicado é se você estiver visualizando essa página com um teclado disponível é precionar as tecla Ctrl com a tecla - (menos) ou a tecla + (mais).

Atenção!

Não utilizar esses recursos diretamente em imagens, sujeito a resultados inesperados

Exemplo de erro:

<img src="foto.png" alt="" class="container" />
