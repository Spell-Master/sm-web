Aplicando a outras tags

Indepedente de uma tag "button" o efeito será aplicado indiferente, portanto qualquer tag pode receber esses padrões.

a.btn-default
div.btn-success

div.btn-info

<a class="btn-default">a.btn-default</a>
<div class="btn-success">div.btn-success</div>
<p class="btn-info">div.btn-info</p>

Personalizado

Aqui usamos a propriedade btn que faz com que o elemento assuma as mesmas características dos botões já apresentados, então usamos as propriedades de plano de fundo para personalizar o botão da forma que desejarmos.

<button class="btn bg-brown bg-dark-orange-hover text-light">
    btn bg-brown bg-dark-orange-hover
</button>
