Utilização

Adicione os arquivos necessários

<link href="Accordion.css" rel="stylesheet" type="text/css"/>
<script src="Accordion.js" type="text/javascript"></script>

Crie a estrutura html

<button class="acc-button">Botão</button>
<div class="acc-container">
    Conteúdo
</div>
<button class="acc-button">Botão</button>
<div class="acc-container">
    Conteúdo
</div>
<button class="acc-button">Botão</button>
<div class="acc-container">
    Conteúdo
</div>
É importante que siga essa arquitetura:
  • Primeiro o botão
  • O próximo elemento deve ser o conteúdo

Inicie declarado em um javascript ao final do documento

<script>
    var acc = new Accordion();
</script>

Método

É possivel ordenar que algum dos elementos se expanda por ação externa utilizando o método forceOpen().

Bastando apenas fornecer como parâmetro o numero do elemento a ser expandido.

Exemplo:

var acc = new Accordion();
acc.forceOpen(2); // Expande o segundo elemento
