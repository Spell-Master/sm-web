width

Largura do elemento

recursion

Esse método é recursivo outros elementos

jsd('#exemplo').width(); // Executa apenas em <div id="exemplo">
jsd('.exemplo').width(); // Executa apenas no primeiro <div class="exemplo">

jsd('#exemplo').width(true); // Executa apenas em <div id="exemplo">
jsd('.exemplo').width(true); // Executa apenas no primeiro <div class="exemplo">

jsd('#exemplo').width('100px'); // Executa apenas em <div id="exemplo">
jsd('.exemplo').width('100px'); // Executa em todos <div class="exemplo">
O retorno da largura é um numero inteiro com base nos pixel's computados
