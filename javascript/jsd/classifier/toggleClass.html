toggleClass

Alterna valores na propriedade do atributo class

recursion

Esse método é recursivo outros elementos

jsd('#exemplo').removeClass('foo'); // Executa apenas em <div id="exemplo">
jsd('.exemplo').removeClass('foo'); // Executa em todos <div class="exemplo">
NOTA*

Se houver mais de um valor na propriedade ou se já existir serão retirados caso contrário será adicionado

<div id="exemplo">exemplo</div>
Alternar apenas um valor 
jsd('#exemplo').removeClass('class-a')
Alternar vários valores 
jsd('#exemplo').removeClass('class-a class-b')
