addClass

Adiciona novos valores na propriedade do atributo class

recursion

Esse método é recursivo outros elementos

jsd('#exemplo').addClass('foo'); // Executa apenas em <div id="exemplo">
jsd('.exemplo').addClass('foo'); // Executa em todos <div class="exemplo">
<div id="exemplo">exemplo</div>
