serialize

Obtem e codifica quaisquer valores contidos em elementos de formulário.
A função serialize só irá codificar elementos no formulário que atendam esses requisitos:
  • Deve possuir o atributo name
  • O atributo name não pode está vazio
  • Não pode possuir o atributo disabled

Atenção

Atributos name vazios em tag <form> resulta em erros na sintaxe do HTML. 
<form id="formulario">
    <input type="text" name="text-name" value="valor do text" />

    <!-- Ausência do atributo name (Não será caputado pelo método) -->
    <input type="text" value="sem nome" class="input-default" />
    <!-- Ausência do atributo name (Não será caputado pelo método) -->
    <input type="text" name="disabled" value="disabled" disabled="" class="input-default" />

    <textarea name="textarea-name">valor do textarea</textarea>

    <select name="select-name">
        <option value="">select</option>
        <option value="valor do select a" selected="">opção a</option>
        <option value="valor do select b">opção b</option>
        <option value="valor do select c">opção c</option>
    </select>

    <input type="checkbox" name="checkbox-name[]" value="valor do checkbox a" checked="" />
    <input type="checkbox" name="checkbox-name[]" value="valor do checkbox b" checked="" />

    <input type="radio" name="radio-name" value="valor do radio a" checked="" />
    <input type="radio" name="radio-name" value="valor do radio b" />
    <button type="button" value="valor do button">button</button>
</form>
<script>
    jsd('#formulario').serialize();
</script>
