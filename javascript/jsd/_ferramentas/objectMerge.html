objectMerge

Unde dois objetos em
Caso haja índices duplicados, prevalecerá os dados do segundo objeto 
var pessoa_A = {
    pessoa: 'João',
    filho: 'joãozinho',
    filha: 'teresa',
    idade: 30
};
var pessoa_B = {
    pessoa: 'Maria',
    filha: 'mariazinha',
    salario: 3000
};

jsd.objectMerge(pessoa_A, pessoa_B);
