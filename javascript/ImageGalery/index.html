ImageGalery.js

O ImageGalery exibe as imagens fora do contexto da página como demonstração individual de cada imagem.

Adicionando assim uma galeria sobre quais imagens serão mostradas, na parte inferior da tela, quando clicado sobre alguma altera a imagem principal de exibição.

Clique sobre alguma imagem para iniciar.

Apresentação:

Exemplo Prático

<img src="imagem_a.jpg" alt="" data-galery="" />
<img src="imagem_b.jpg" alt="" data-galery="" />
<img src="imagem_c.jpg" alt="" data-galery="" />
<script>
    var exemplo = new ImageGalery();
</script>

Utilização

Adicione o arquivos necessários:

<script src="ImageGalery.js" type="text/javascript"></script>
<link href="ImageGalery.css" rel="stylesheet" type="text/css" />

Adicione os arquivos de imagens necessários, exemplo:

<img src="imagem_a.jpg" alt="" data-galery="" />
<img src="imagem_b.jpg" alt="" data-galery="" />
<img src="imagem_c.jpg" alt="" data-galery="" />

Perceba o uso do atributo data-galery. Este atributo é o gatilho de dispara a iniciação do script, qualquer imagem que o possua será alvo das funções.

Caso alguma não tenha esse etributo não será carregada pelo ImageGalery.js

O próximo passo é instanciar o ImageGalery, essa etapa é adequada ser iniciado no final do documento html antes do fechamento da tag </body>

No entanto também pode-se iniciar logo após a renderização dos elementos.

<script>
    var exemplo = new ImageGalery();
</script>

*NOTA:

Para reduzir o custo do desempenho, o ImageGalery não irá verificar se todas as imagens estão devidamente carregadas, portanto incoerências aleatórias podem ocorrer devido a falhas de carregamento, imagens muito pesadas ou ataque cross-script.

O processo de checagem pode ser realizado antes de criar a instância se achar que seja necessário.

Métodos Extras

reload

Este método irá recarregar as imagens pertecentes as funções, caso venha a remover ou adicionar imagens você poderá usar-lo para redefinir o ImageGalery a condição atual.

<script>
    var exemplo = new ImageGalery();
    exemplo.reload(); // Recarrega tudo.
</script>

Métodos de Extras

elements

Este método irá retonar todas imagens pertecentes as funções.

<script>
    var exemplo = new ImageGalery();
    exemplo.elements(); // NodeList das imagens.
</script>
