Rolagem

Força a criação de uma rolagem quando um elemento interno for maior que o externo impedindo o vazamento.

over-auto

Permite na vertical e horizontal

Lorem ipsum dolor sit amet.... 
<div class="over-auto box-y-200 margin-auto">
    <div class="box-xy-900 bg-red text-white">
        Lorem ipsum dolor sit amet....
    </div>
</div>

Força a remoção de uma rolagem quando um elemento interno for maior que o externo impedindo o vazamento.

over-not

Impedirá que a rolagem apareça.

    <div class="box-xy-900 bg-red text-white">

    </div>
</div>
