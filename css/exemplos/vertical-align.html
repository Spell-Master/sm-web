Alinhamento vertical e horizontal

A propriedade vertical-wrap alinha elementos filhos centro de elemento alvo na horizontal e vertical.

Mas é necessário que o mesmo possua uma altura de referência.

vertical-wrap 
<div class="vertical-wrap box-y-200 bg-black text-white">
    <span class="font-max">vertical-wrap</span>
</div>

Todo e qualquer elemento perde sua orientação de margens, tudo será centralizado colocando cada elemento ao lado do outro ao centro, mesmo que sejam quebras de linhas como <br />, <p>, <div> etc...

Esse é um recuso ideal para se usado somente em texto, mas caso necessite que os elementos respeitem sua orientação considere usar posições.

Uma tag "DIV"
Outra tag "DIV"

Um parágrafo "P"

<div class="relative box-y-200 bg-black text-white">
    <div class="pos-center">
        <div>Uma tag "DIV"</div>
        <div>Outra tag "DIV"</div>
        <p>Um parágrafo "P"</p>
    </div>
</div>
