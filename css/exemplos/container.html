Propriedades que centralizam e adaptam o conteúdo interno, mesmo que resolução seja menor que o esperado

IMPORTANTE!

Essas propriedades só devem ser usadas como recipientes de outros conteúdos internos.

Tal forma que as mesmas são recomendadas que sejam filhas diretamente da tag <body>.

container - Margens ajustadas até 1920 pixels aproximadamente.

Ideal para full HD

container-min - Margens ajustadas até 480 pixels aproximadamente.

Ideal para smartphone

container-medium - Margens ajustadas até 1024 pixels aproximadamente.

Ideal para monitor genérico 24 polegadas

container-max - Margens ajustadas até 3840 pixels aproximadamente.

Ideal para 4k

</div>
