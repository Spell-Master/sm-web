selectIndex

Manipula o índice selecionado de tag <select>

recursion

Esse método é recursivo outros elementos


jsd('#exemplo').selectIndex(); // Executa apenas em <div id="exemplo">
jsd('.exemplo').selectIndex(); // Executa apenas no primeiro <div class="exemplo">

jsd('#exemplo').selectIndex(2); // Executa apenas em <div id="exemplo">
jsd('.exemplo').selectIndex(2); // Executa em todos <div class="exemplo">
Caso o valor designado para alterar o seletor for maior que as opções disponíveis o seletor não será modificado. 
<select id="exemplo">
    <option value="valor do select a">opção a</option>
    <option value="valor do select b">opção b</option>
    <option value="valor do select c">opção c</option>
</select>
Mostra o índice selecionado 
jsd('#exemplo').selectIndex()
Altera o índice selecionado 
jsd('#exemplo').selectIndex(2)
